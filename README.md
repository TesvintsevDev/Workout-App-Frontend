## Running Locally

Clone project

```bash
  git clone https://github.com/TesvintsevDev/Workout-App-Frontend
```

Go to project directory

```bash
  cd Workout-App-Frontend
```

Install dependencies

```bash
  npm install

```

Start development server

```bash
  npm run dev
```
